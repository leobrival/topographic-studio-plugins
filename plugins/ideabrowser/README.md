# IdeaBrowser Plugin

Product and marketing agents for ideation, strategy, and execution.

## Agents

| Agent | Description |
|-------|-------------|
| accessibility-auditor | Web accessibility audit with RGAA 4.1 |
| ad-creatives | Ad copy and creative concepts |
| blog-strategist | SEO blog content strategy |
| brand-package | Brand identity design |
| content-calendar | 90-day content marketing plan |
| customer-interview-guide | Customer interviews with Mom Test |
| customer-success-playbook | Retention and expansion strategy |
| deep-search | Autonomous research agent |
| email-funnel-system | Email marketing automation |
| email-sequence | 5-email nurture sequences |
| feature-specs | Feature specifications with user stories |
| gamification-design | UX gamification with Octalysis |
| gtm-launch-calendar | 90-day launch timeline |
| gtm-strategy | Go-to-market strategy |
| kpi-dashboard | Metrics and analytics framework |
| landing-page | High-converting landing pages |
| lead-magnet | Lead generation offers |
| market-research-intelligence | Competitive analysis and research |
| mvp-roadmap | 90-day development plan |
| pricing-strategy | Pricing and revenue optimization |
| product-analytics | Product metrics and insights |
| product-designer | UI/UX design systems |
| product-requirements-doc | Complete PRD creation |
| product-strategy-execution | Strategic product building |
| product-technical-specs | Technical specifications |
| referral-program | Viral referral design |
| sales-funnel | Customer journey optimization |
| seo-content | Search-optimized content |
| trend-researcher | Market trends and opportunities |
| tweet-sized-landing-page | Ultra-minimal landing pages |
| user-personas | Customer persona development |

## Usage

Agents are automatically available via the Task tool:

```
Use the landing-page agent to create a high-converting page
```
