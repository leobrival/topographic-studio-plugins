# Product Plugin

Product design and management tools for building user-centric products.

## Overview

The Product plugin provides specialized agents and skills for product design, user research, and design systems development.

## Contents

### Agents

- **product-designer**: Expert in UI/UX design, design systems, brand identity, and gamification. Creates comprehensive style guides and production-ready components.

### Skills

- **user-personas**: Create detailed customer persona cards with motivations, Jobs-to-be-Done framework, behavioral insights, and empathy mapping.

## Getting Started

Use the product-designer agent for:

- Design system creation and maintenance
- Brand identity development
- UI/UX design for web and mobile applications
- Design primitive definition
- Component library design
- Interaction design and specifications
- Gamification strategy using Octalysis framework

Use the user-personas skill for:

- Creating detailed customer personas
- Jobs-to-be-Done analysis
- Customer empathy mapping
- User research documentation
- Persona validation with The Mom Test methodology

## Features

### Design Capabilities

- Design tokens and design primitives
- Component library architecture (Atomic Design)
- Brand consistency enforcement
- Typography systems
- Color palette management
- Spacing and layout systems
- Responsive design patterns
- Accessibility integration (WCAG)

### User Research

- Persona creation with behavioral insights
- JTBD (Jobs-to-be-Done) framework
- Customer awareness stages
- 30 Elements of Value analysis
- Empathy mapping
- Mom Test validation principles

## Configuration

The plugin is configured in `.claude-plugin/plugin.json` with agents and skills auto-discovered from the plugin structure.
