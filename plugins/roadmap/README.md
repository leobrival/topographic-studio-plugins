Technical engineering agents for software development and architecture.

| ai-engineer | LLM integration and RAG systems |

| data-engineer | Data pipelines and ETL |
| data-scientist | Statistical analysis and ML |
| devops-engineer | CI/CD and infrastructure automation |
| docker-containerization | Docker and container orchestration |
| frontend-developer | Modern web applications |
| fullstack-engineer | End-to-end application architecture |
| git-workflows | Version control and branching strategies |
| golang-engineer | Go development and microservices |
| javascript-mastery | Advanced JavaScript patterns |
| kubernetes-orchestration | Kubernetes cluster management |
| linux-sysadmin | Server management and scripting |
| machine-learning-engineer | Model training and MLOps |
| mlops-specialist | ML pipelines and deployment |
| nodejs-architect | Node.js scalable systems |
| prompt-engineer | LLM optimization and prompting |
| python-developer | Python frameworks and patterns |
| react-specialist | React hooks and performance |
| rust-systems | Rust systems programming |
| software-architect | System design and patterns |
| system-design | Scalable distributed systems |
| terraform-iac | Infrastructure as Code |
| typescript-expert | Advanced TypeScript patterns |

## Usage

Agents are automatically available via the Task tool:

```
Use the software-architect agent to design the system architecture
```
